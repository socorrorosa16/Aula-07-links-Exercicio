Introdução do MVC

O padão MVC (Model view controlle) é uma arquitetura software utilizada no desenvolvimento de aplicaç~es, para separar a lógica de negócios, a interface do usuário e o controle de dados

Componentes do MVC

  • Model(Modelo)- Representa os dados e a logica de negócio de aplicação
  • View(Visão)-Responsavel pela interfacecom o usuário e exibição dos dados.
  • Controlle(Controlador)Intermedia a comunicação entre o model e o valor
